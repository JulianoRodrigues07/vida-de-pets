# 🐾 Vida de Pets — Site Oficial

Site completo desenvolvido para o petshop **Vida de Pets**, localizado em Passo Fundo - RS. O projeto nasceu de uma demanda real: criar uma presença digital profissional para o negócio, com sistema de agendamento online e painel administrativo.

> Projeto fullstack desenvolvido com Next.js App Router, TypeScript, Tailwind CSS v4, Prisma e NextAuth.js.

---

## ✨ Funcionalidades

### Área pública
- **Home** com hero animado, colunas de fotos em scroll infinito e avaliações reais do Google
- **Página de Serviços** com bento grid interativo e detalhamento de cada serviço
- **Página Sobre** com história do petshop, seção da fundadora e galeria de fotos em carrossel
- **Página de Contato** com mapa do Google Maps integrado, WhatsApp e Instagram
- **Página Creche & Hotel** exclusiva com galeria interativa, diferenciais e horários
- **Sistema de Agendamento** com bloqueio de horários já ocupados em tempo real

### Área administrativa
- **Login protegido** com senha (NextAuth.js + Credentials Provider)
- **Painel de agendamentos** com listagem e atualização em tempo real
- **Confirmação via WhatsApp** com mensagem pré-preenchida com os dados do cliente
- **Reagendamento via WhatsApp** com mensagem personalizada
- **Cancelamento com confirmação** em dois passos para evitar erros

## 🧠 Aprendizados

Este projeto foi desenvolvido como parte do meu portfólio, com foco em aprender e aplicar:

- Arquitetura de projetos Next.js com App Router
- Diferença entre Server Components e Client Components
- Criação de API Routes no Next.js
- Modelagem de banco de dados com Prisma ORM
- Autenticação com NextAuth.js (Credentials Provider)
- Proteção de rotas com Middleware
- Boas práticas de acessibilidade (focus-visible, aria-label, lang)
- Componentização e organização de código em React
- Animações com Framer Motion (entrance, scroll-triggered, infinite)
- CSS avançado (infinite scroll vertical, gradientes, keyframes)
- Integração com Google Maps via iframe
- Geração de QR Code com qrcode.react
- Deep links do WhatsApp com mensagens pré-preenchidas
